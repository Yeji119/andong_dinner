# 안동 맛집 소개

이 프로젝트는 안동의 맛집을을 소개하는 웹페이지입니다.

HTML과 CSS를 사용하여 기본적인 웹페이지를 제작하였으며,  
시맨틱 태그를 이용해 구조화하고, 각 맛집에 대한 상세 페이지 링크를 제공합니다.

## 주요 기능
- 안동 지역의 맛집 목록 제공
- 각 맛집 사진 클릭 시 상세 페이지로 이동
- 표를 활용한 맛집 정보 요약
- 구글 지도 임베딩을 통한 위치 확인
